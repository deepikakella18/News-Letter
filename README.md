Newsletter Sign Up

A simple and responsive Newsletter Sign Up web page built using HTML and CSS. This project provides a clean interface where users can enter their email address to subscribe to a newsletter. It is designed to demonstrate the fundamentals of web forms, HTML structure, CSS styling, and basic input validation.

 Features
Clean and responsive design
Email input with HTML validation
Simple and user-friendly interface
Styled Subscribe button with hover effect
Beginner-friendly project
 Technologies Used

CSS3
